# 📊 Dashboard Projects Included

# 📉 1. Global Layoffs Analysis Dashboard

## 🔍 Project Description

This dashboard analyses global layoffs data to identify trends across companies, industries, countries, and time periods. Interactive visualisations highlight workforce reduction trends and economic impact across various sectors.

---

## 📈 Key Features

* Total layoffs KPI tracking
* Layoffs trends over time
* Top companies by layoffs
* Industry-level analysis
* Geographic analysis by country
* Interactive filtering and drill-down functionality

---

## 🔍 Key Insights

* Certain industries experienced significantly higher layoffs than others
* Layoff activity peaked during specific periods
* A small number of companies contributed heavily to total layoffs
* Geographic patterns reveal concentration in specific countries

---

# 🏡 2. AirBnB Revenue & Pricing Analysis Dashboard

## 🔍 Project Description

This dashboard analyses AirBnB listing data to explore how factors such as location, bedroom count, and pricing influence revenue generation and listing performance.

---

## 📈 Key Features

* Revenue analysis by location
* Average pricing by bedroom count
* Geographic distribution of listings
* Interactive filters for neighbourhoods and room types
* Comparative pricing visualisations

---

## 🔍 Key Insights

* Location strongly impacts listing revenue and pricing
* Larger properties generally generate higher revenue
* Certain neighbourhoods outperform others significantly
* Pricing trends vary based on room type and property size

# 📋 3. Data Professionals Survey Dashboard

## 🔍 Project Description

This dashboard analyses survey data collected from data professionals across multiple countries and job roles. The project explores salary trends, programming language preferences, work-life balance, and overall job satisfaction within the data industry. 

---

## 📈 Key Features

* Average salary analysis by job title
* Favourite programming language breakdown
* Country distribution of survey participants
* Work-life balance analysis
* Salary satisfaction analysis
* Interactive dashboard visuals and filtering

---

## 🔍 Key Insights

* Data Scientists reported the highest average salaries
* Python emerged as the most popular programming language among respondents
* Salary satisfaction was lower than overall work-life balance ratings
* Survey responses were distributed across multiple countries and data-related professions
* Data Analysts and Data Engineers formed a large portion of respondents
